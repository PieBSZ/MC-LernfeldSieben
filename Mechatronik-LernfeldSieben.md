### Blockschaltbild

Eine Regelung wird meist schematisch als **Blockschaltbild** anhand der Struktur eines **Regelkreises** dargestellt.

**AUFGABE**

``` ascii
+------------+   +---------+   +-------+
|            |   |         |   |       |
| Förderband +-->| Roboter +-->| Fräse |
|            |   |         |   |       |
+------------+   +---------+   +-------+
```
